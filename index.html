This is a tool based on convolutional neural networks for text similarity measurement.

There could be rapid code changes, since we are still working on codes integration and cleanup right now,

Dependencies: 1. The code requires the Torch deep learning library. Please install it first. We also requires 'nngraph' package and 'lfs' package. 2. This requires Standford's Glove word embeddings. Please download this word embeddings (we will provide scripts for this)

This repository already comes with SICK data and MSRVID data.

Then, simply type: ``th trainSIC.lua'' to get our convolutional neural network model trained on the SICK data set.

More instructions/code updates to be coming. You can check back this page later.
